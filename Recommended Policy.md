# Recommended Policy

- Allow everyone to create personal/private low-risk agents only.
- Allow trained/approved makers to create team/department agents in managed environments.
- Require central IT / engineering / platform review for any agent that is:
   - shared org-wide,
   -  connected to business systems,
   - uses sensitive data,
   - performs actions,
   - or is considered production-critical.

That is very close to what Microsoft is pushing as the safe enterprise model.
